# AI能力：生成标题

## 触发词
当Jason说"生成标题"或"标题"时触发

## 工作流程
1. **读取上下文**
   - 如果Jason刚完成文稿，基于文稿内容
   - 如果Jason指定了文稿，读取指定文件

2. **参考方法论**
   - 读 `03-方法论/标题规律.md`（如果有数据积累）
   - 遵循小红书标题特点：
     - 15字以内最优
     - 包含数字、emoji、符号增加吸引力
     - 制造悬念或痛点共鸣
     - 避免标题党（与内容一致）

3. **生成3个选项**
   - 选项1：痛点型（直击用户痛点）
   - 选项2：利益型（强调效果/好处）
   - 选项3：好奇型（制造悬念）

4. **说明逻辑**
   - 每个标题为什么这么写
   - 预期吸引的人群

## 示例输出
```
✅ 为《冬季护肤保湿》生成3个标题选项：

【选项1】痛点型
干皮冬天真的会死😭这瓶救命！
- 逻辑：直击干皮冬季痛点，emoji增强情绪
- 目标人群：干皮、混干皮

【选项2】利益型
LA MER面霜实测🧴滋润度拉满！
- 逻辑：品牌+效果承诺，实测增加可信度
- 目标人群：追求高端护肤的人群

【选项3】好奇型
码农小经理的冬季护肤狠货💆🏻♀️
- 逻辑：人设+场景，引起共鸣和好奇
- 目标人群：职场女性、北美华人

推荐：选项1（痛点最强）或选项3（人设鲜明）
```

## 注意事项
- 标题必须真实反映内容，不夸张
- emoji使用自然，参考风格指南
- 如果方法论里有数据支持的规律，优先遵循
- 提供多样化选择，让Jason有选择空间
