# AI能力：记录选题

## 触发词
当Jason说"记录选题"或"记一下选题"时触发

## 工作流程
1. 识别Jason提到的选题方向/想法
2. 追加到 `01-选题/想法记录.md`
3. 格式：`日期 | 选题方向 | 简要描述`
4. 确认记录完成

## 示例对话
```
Jason: "记录选题：冬季护肤保湿，重点推LA MER面霜"

AI: 
- 识别选题：冬季护肤保湿
- 产品：LA MER面霜
- 追加到想法记录.md
- 回复："✅ 已记录选题：冬季护肤保湿（LA MER面霜）"
```

## 注意事项
- 如果Jason提供的信息不完整，主动追问补充
- 如果发现类似选题已存在，提醒Jason避免重复
