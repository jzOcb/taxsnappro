# 个人写作风格指南 / Personal Writing Style Guide

## 博主人设 / Persona
- 北美全职码农小经理 + 副业博主
- 爱淘全球小众护肤，尤其是欧洲小众贵妇牌
- 注重性价比与功效，懂高科技护肤成分
- 经常旅行，有海外购物经验

## 开场风格 / Opening Patterns

### 个人经历切入
```
作为一个北美全职码农小经理，外加做副业，长时间面对屏幕加熬夜...
```

### 情绪共鸣开场
```
宝子们！又到了年底最开心的囤货时刻啦✨
谁能懂啊！[震撼经历]...
```

### 旅行/场景引入
```
刚从[地点]回来，现在还沉浸在震撼里没缓过来😭
```

## 常用口语表达 / Signature Phrases

### 高频词汇
- 宝子们、姐妹们
- yyds、绝绝子
- 闭眼冲、闭眼入
- 直接拉满、性价比拉满
- 狠货、狠活
- 救星、续命
- 在线（效果在线、滋润度在线）
- 妥妥的
- 真的会谢

### 产品描述用语
- 质地：丝绒绸缎质地、蛋清质地、奶油般、绵密的乳液状
- 吸收：一秒推开、秒吸收、融入肌肤
- 效果：肉眼可见、第二天起来、水润透亮
- 推荐：干皮可以/混合皮建议/油皮谨慎

### 转折/强调
- 不过...
- 说真的...
- 真的是...
- 就是那种...

## 产品介绍结构 / Product Description Structure

### 标准模板
1. **品牌背景**（简短，突出小众/高端/科技感）
2. **质地描述**（具体触感，如"像淡奶油一样顺滑"）
3. **核心成分**（用通俗语言解释功效）
4. **使用方法**（针对不同肤质给建议）
5. **个人体验**（真实感受+效果）
6. **适合人群**（明确推荐/不推荐）

### 示例
```
首先是这个[品牌][产品]，[品牌背景1句话]。
质地是[具体描述]，上脸[使用感受]。
核心成分是[成分名]，主打[功效]。
我喜欢[使用方法]，[个人效果]。
[肤质]可以/建议[用法]。
```

## 脚本结构 / Script Structure

### 短视频 (30秒-1分钟)
```
【开场】(0:00-0:05)
- 1句话人设/场景引入
- 直接点出痛点或主题

【主体】(0:05-0:25)
- 1-2个产品核心亮点
- 质地+效果+个人感受

【收尾】(0:25-0:30)
- 简短总结或推荐语
```

### 中长视频 (3-10分钟)
```
【开场】(0:00-0:45)
- 营造氛围，引入主题
- 说明视频内容概览

【产品分段】(按产品数量分配时间)
- 每个产品1-2分钟
- 按上述产品介绍结构展开

【结尾】(最后30-60秒)
- 总结升华（不只是产品，是生活态度）
- 互动引导（评论区告诉我/喜欢点赞收藏）
```

## Emoji 使用习惯 / Emoji Usage
- ✨ 惊喜/推荐
- 😭 震撼/感动/后悔
- 🤯 难以置信
- 💆🏻♀️ 护肤相关
- 🧴 产品
- 🌍 旅行
- 🌟 亮点/推荐

## 标签风格 / Hashtag Style
- 混合使用话题标签
- 产品相关 + 场景相关 + 功效相关
- 例：#小众护肤 #欧洲护肤品 #抗老精华 #以油养肤 #旅行护肤必备

## 注意事项 / Important Notes
- 保持真实感，分享真实使用体验
- 适当提及缺点增加可信度
- 成分解释要通俗易懂，不要太专业
- 针对不同肤质给出具体建议
- 结尾要有互动引导
