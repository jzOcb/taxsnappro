# AI能力：生成文稿

## 触发词
当Jason说"写文稿"或"深化选题：[选题名]"时触发

## 工作流程
1. **检索素材库**
   - 查 `02-素材/金句库.md` — 有没有可复用的hook句式？
   - 查 `02-素材/产品库.md` — 有没有相关产品信息？
   - 查 `01-选题/已发布/` — 有没有类似选题？

2. **建议复用**
   - 如果找到相关内容，告诉Jason："你之前做过[相关选题]，要不要复用？"
   - 展示可复用的内容片段

3. **生成文稿**
   - 如果没有可复用的，或Jason决定创作新的
   - 严格遵循 `03-方法论/风格指南.md` 的风格
   - 按短视频或长视频结构生成
   - 保存到 `01-选题/待做/[选题名].md`

4. **确认**
   - 告诉Jason文稿已保存位置
   - 询问是否需要生成标题

## 示例对话
```
Jason: "写文稿：冬季护肤保湿"

AI:
1. 检索素材库...
   - 产品库找到：LA MER面霜信息
   - 金句库找到：开场句式"宝子们！又到了XX时刻"
   - 已发布找到：类似选题《秋季补水》

2. "你之前做过《秋季补水》选题，框架可以复用。这次重点换成LA MER？"

3. [Jason确认后] 生成文稿，遵循风格指南...

4. "✅ 文稿已保存到 01-选题/待做/冬季护肤保湿.md，需要生成标题吗？"
```

## 注意事项
- 先检索再创作，避免重复造轮子
- 严格遵循风格指南中的人设和表达习惯
- 产品描述必须准确（从产品库获取）
- 不确定的信息不要编造，问Jason
