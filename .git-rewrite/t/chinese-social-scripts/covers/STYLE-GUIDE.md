## 核心认知

**封面 = 视频截帧/照片 + 文字叠加**
不是纯设计图，不是白底排版，不是装饰画。

## 文字风格

### 主标题
- **超大粗体** — 占画面宽度 50-80%
- **描边/阴影** — 白字黑描边，或彩字黑描边，确保任何背景都能看清
- **字数 ≤ 6字** — "年度封神消费"、"直播打仗"、"买少买精"、"生日礼物"
- **混合字号** — 关键词放大，次要词缩小（"最值的**6笔**！"）

### 文字颜色
- 白色 + 黑描边（最常见，万能）
- 红色/橙色 + 黑描边（强调、促销感）
- 黄色/金色 + 黑描边（高级感）
- 绿色（Jason 账号常用，如"护肤舒适区"）

### 描边参数（技术）
- 描边宽度: 字号的 3-5%
- 描边颜色: 纯黑或深灰
- 可选: 文字阴影（向右下偏移 2-3px）

## 标题公式（直击痛点）

| 公式 | 示例 | 效果 |
|------|------|------|
| 数字+品类 | "回购前5面膜"、"TOP5" | 明确预期 |
| 痛点+方案 | "多维打击告别黄黑"、"油痘皮夏日准备" | 引共鸣 |
| 年龄+需求 | "30+油皮抗老"、"30+抗皱刚需" | 精准定位 |
| 反差/情绪 | "平价也能出奇迹"、"买少买精" | 好奇心 |
| 疑问句 | "女孩子为什么要努力？" | 互动感 |
| 场景+推荐 | "长途飞行不烂脸攻略" | 实用感 |

## 布局模式

### 模式A: 底部大字（最常见）
- 人物占上半部分
- 大标题在底部 1/3
- 适合: 出镜类内容

### 模式B: 散落标签
- 人物居中
- 多个小标签散布在画面各处（#油敏痘 #氧化老化）
- 适合: 教程/干货类

### 模式C: 左侧竖排
- 文字竖排在左侧
- 人物/产品在右侧
- 适合: 开箱/合集类（如"年度新品 TOP5"）

### 模式D: 对比/分屏
- 画面分左右或上下
- 文字在中间
- 适合: 对比评测类（如"法国🇫🇷 vs 瑞士🇨🇭"）

## 装饰元素（少量点缀）
- ✨ 闪光/星星（小，1-3个）
- ❤️ 爱心 emoji
- 圆圈/箭头指向产品
- 英文小字点缀（如 "Night Routine", "Sharing", "January shopping haul"）
- **绝对不要**: 大面积花纹、渐变背景、礼物盒插画等

## 技术实现方向

封面生成器应该是一个 **文字叠加工具**：
1. 输入: 照片/视频截帧
2. 添加: 大字标题（带描边）
3. 添加: 可选的小标签/hashtag
4. 添加: 可选的 emoji 装饰
5. 输出: 1242x1660 封面图

不需要: 生成背景、画装饰元素、做复杂排版
