# 小红书平台规格 / RedNote Specifications

## 字数限制
| 内容 | 限制 |
|------|------|
| 标题 | 20字 |
| 正文 | 1000字 |
| 标签 | 单个标签20字，建议5-10个 |

## 图片要求
- 数量：最多18张
- 比例：推荐3:4（竖图更佳）
- 质量：高清，美观

## 视频要求
- 时长：最长15分钟
- 比例：9:16竖屏最佳
- 封面：吸引人的第一帧

## 发布建议

### 最佳发布时间
- 工作日：12:00-14:00, 20:00-22:00
- 周末：10:00-12:00, 15:00-17:00, 20:00-23:00

### 标题技巧
- 使用数字（「5个方法」）
- 加emoji增加视觉吸引
- 痛点+解决方案
- 制造好奇心

### 标签策略
- 热门大标签 + 精准小标签
- 相关性高于数量
- 关注近期热门话题
