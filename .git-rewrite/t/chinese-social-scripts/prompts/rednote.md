# 小红书文案提示词 / RedNote Prompts

## 平台特点
- 图文为主，视觉美感重要
- 种草文化，真实分享
- 标题决定点击率
- 字数限制：标题20字，正文1000字

---

## 种草笔记

```
请写一篇小红书种草笔记：

产品/主题：[名称]
核心卖点：[1-3个亮点]
使用场景：[什么情况下用]

要求：
- 标题吸引人，带emoji
- 正文真实有细节
- 结尾互动引导
- 5-10个相关标签
```

## 标题生成

```
为以下内容生成5个小红书标题：

主题：[主题]

公式参考：
1. 数字+结果：「7天亲测有效」
2. 身份+痛点：「学生党必看」
3. 反常识：「后悔没早知道」
4. 场景+方案：「约会前急救」
5. 情绪共鸣：「真的会谢」
```

## 探店测评

```
写一篇探店笔记：

店铺/产品：[名称]
价格：[价格]
亮点：[好的方面]
不足：[改进之处]
评分：[X/10]

保持客观真实。
```

## 攻略笔记

```
写一篇攻略笔记：

主题：[主题]
核心要点：[列表]

格式：分点清晰，emoji辅助阅读，信息实用。
```
